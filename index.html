Keynobi

2014 DataWeek Hackathon Winner!

Keynobi uses the HP IDOL OnDemand facial recognition API as authentication for password management. The application includes a web application and a Google Chrome extension; users may manage their login information via the web application and use the Chrome extension to log into websites, all using facial recognition via a computer's webcam. The IDOL OnDemand API is not yet available for public use.

Simplicity

This project emulates a rubegoldberg machine, or series or random complex events to do something simple such as telling the time. We used a sinatra framework and had a master controller that plugged in JavaScript modules after a certain amount of time. The website also connected to a Wemo to bring the Rube Goldberg website into reality so you might not get the whole effect. I was responsible for the 3D cube module and the webcam photos module as well as the simple index page design.
